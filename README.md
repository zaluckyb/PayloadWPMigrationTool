# Payload WP Migration Tool

This repository contains early scaffolding for a WordPress export plugin and a Node.js/TypeScript migration orchestrator intended to move WordPress content into a Payload CMS 3.53 application.

## Structure

- `wp-plugin/` – placeholder WordPress plugin exposing a minimal `/site` REST endpoint.
- `orchestrator/` – TypeScript CLI skeleton with stubbed commands.

> **Note**: This is a preliminary scaffold. Full implementation of all migration features, authentication, real-time sync, and schema generation is still required.
